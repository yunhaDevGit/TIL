## httpd

- HTTP Protocol을 지원하는 daemon
- 웹 서버의 백 그라운드에서 실행되어 들어오는 서버 요청을 대기하는 소프트웨어 프로그램
- 이 데몬은 자동으로 요청에 응답하여 HTTP를 사용해 인터넷을 경유, 하이퍼텍스트, 멀티미디어 문서들을 서비스한다.
  - Web Server를 구축한다 = httpd를 수행한다



#### Apache(80)

- 아파치 소프트웨어 재단에서 만든 **웹 서버 프로그램**
- 이미지, html, css, javascript와 같은 정적 페이지 처리



#### Tomcat(8080)

- **WAS(Web Application Server)**

- 자바 서블릿 실행 -> JSP 코드가 포함된 웹 페이지를 만들어준다

  그 결과를 다시 html로 재구성해서 아파치한테 돌려준다

- 톰캑 자체적으로 보유하고 있는 웹 서버로 동작할 수 있다

  아파치, 넷스케이프 엔터프라이즈 서버 등 다른 웹서버와도 함께 사용할 수 있다.

- 사용자 데이터 처리를 하는 동적 페이지 처리
