# 구글 로그인



### OAuth

- 별도의 회원가입 없이 로그인을 제공하는 플랫폼의 아이디만 있으면 서비스를 이용할 수 있도록, 외부 서비스에서도 인증을 가능하게 하고, 그 서비스의 API를 이용하게 해주는 것

  - ex) 네이버, 페이스북, 구글, 카카오 로그인
